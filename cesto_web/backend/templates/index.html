Endpoints de la API

GET /api/dashboard

Obtiene todos los datos del sistema en un solo endpoint.

Ejemplo: /api/dashboard

GET /api/fill-levels

Obtiene los niveles de llenado actuales de todos los compartimentos.

Ejemplo: /api/fill-levels

GET /api/statistics

Obtiene las estadísticas de clasificación de residuos.

Ejemplo: /api/statistics

GET /api/system-status

Obtiene el estado actual del sistema.

Ejemplo: /api/system-status
